## Descrição do Projeto

Este é um aplicativo mobile desenvolvido em React Native com Expo que apresenta informações sobre tecnologia e inovação digital. O projeto foi desenvolvido como trabalho acadêmico, demonstrando conhecimentos em desenvolvimento mobile, interface responsiva e gerenciamento de estado em React Native.

O aplicativo oferece uma interface intuitiva com navegação por abas, exibição de listas com imagens, informações detalhadas e interatividade através de modais.
